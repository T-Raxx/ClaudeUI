Qué se carga y cuándo

La base se baja sola. El rage no: hay que pedirlo en Settings, avisa de los riesgos y chequea que tu ejecutor tenga las funciones antes de cargar.

ArchivoQué esCargaHooks
Loader.luaBaja el principal—no
RivalsMain.luaESP, aimbot, triggerbot, chams, HUD, configssiempreno
RivalsUI.luaLa librería de UI (Drawing)siempreno
RivalsVisuals.luaMundo: lighting, fog, tinte, cielo, climasiempreno
RivalsCombat.luaCámara y efectos, vía API del juegosiempreno
RivalsRage.luaRage: spoof de posición, disparo interno, welda manosí

Requiere Drawing API y game:HttpGet. El módulo rage además necesita hookmetamethod, checkcaller, getgenv, setthreadidentity y sethiddenproperty.
